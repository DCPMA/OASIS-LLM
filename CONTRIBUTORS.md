## How to be added

If you have contributed (code, ideas, study design, analysis, documentation,
issue triage, or anything else that materially shaped the project) and would
like to be credited, open an issue or pull request adding yourself to this
file with:

- Name (or pseudonym).
- ORCID, GitHub handle, personal site, or other links — whichever you'd like
  publicly associated with this work.
- The CRediT roles that best describe your contribution. The full list is in
  the [taxonomy](https://credit.niso.org/contributor-roles-defined/).

For substantive research contributions we will also add you to
[`CITATION.cff`](CITATION.cff) so you appear in the GitHub *Cite this
repository* button and any downstream citations.
