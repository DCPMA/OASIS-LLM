# **Emotional Reactivity in Multimodal Large Language Models: A Psychometric Benchmarking Study**

**Research question.** To what extent do multimodal LLM-generated valence and arousal ratings align with human normative ratings for validated emotion-eliciting visual stimuli from the Open Affective Standardized Image Set (OASIS)?

The goal is **methodological and explorative**: Employing validated psychometrics and rigorous experimental design to examine whether multimodal large language models (LLMs) generate affective responses to visual stimuli that are structurally comparable to normative human data.

## What this project is

- **A pioneering and rigorous design** to examine LLMs pattern matching with emotion-eliciting materials.
- **A reproducible run harness** built on DuckDB + LiteLLM + asyncio: configs are YAML, runs are idempotent, results are queryable.
- **A discrepancy log.** We are explicit about where this replication diverges from the human protocol — see [Discrepancies](/discrepancies).

## What this project is **not**

- Not a leaderboard. We do not aim to publish model rankings.
- Not a claim that LLMs experience any emotion. The ratings are model behaviour; what they correspond to in the model is an open question.
- Not yet validated. Reliability and validity of the LLM performance vs. the human norms are still under analysis.
