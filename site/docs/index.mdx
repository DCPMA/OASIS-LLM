---

OASIS-LLM is a small experimental harness that takes the **Open Affective Standardized Image Set (OASIS)** — a 900-image affective stimulus set normed by 822 MTurk workers on 7-point valence and arousal scales — and replays the rating procedure against modern vision-language models.

The goal is **methodological**: figure out what it takes to get behaviorally meaningful ratings out of an LLM (prompt fidelity, sampling variance, cost, reasoning capture) and how those ratings compare to the human norms.

## What this project is

- **A faithful re-implementation** of the OASIS rating procedure using paper-verbatim system prompts (image-centered instructions only, matching the main study).
- **A reproducible run harness** built on DuckDB + LiteLLM + asyncio: configs are YAML, runs are idempotent, results are queryable.
- **A discrepancy log.** We are explicit about where this replication diverges from the human protocol — see [Discrepancies](/discrepancies).

## What this project is **not**

- Not a benchmark leaderboard. We do not publish model rankings.
- Not a claim that LLMs feel valence or arousal. The ratings are model behavior; what they correspond to in the model is an open question.
- Not yet validated. Reliability and validity vs. the human norms are still being characterized.
